# Hospital Performance Dashboard (Excel Project)

This project is a complete Excel-based analytics solution designed to monitor and analyze key hospital performance metrics. It includes a clean and interactive dashboard supported by pivot tables for deeper analysis.

The goal of this project is to convert raw healthcare data into actionable insights for hospital decision-making, capacity planning, and operational performance tracking.
